# Archive

This folder contains historical documentation from the development of go-ffmpeg-hls-swarm. These documents are preserved for reference but may be outdated or superseded by current documentation.

## Contents

### design-docs/

Pre-implementation design documents that describe features before they were built. These capture the original thinking and trade-offs considered.

### implementation-logs/

Work-in-progress tracking documents that recorded development progress. Useful for understanding how features evolved.

### implementation-plans/

Step-by-step implementation plans that outlined how features would be built. May differ from actual implementation.

### analysis/

Technical investigations and debugging sessions. Useful for understanding complex issues that were resolved.

### defects/

Bug tracking documents that recorded issues and their resolution.
